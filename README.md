# Quiz interativo
Essé um projeto é uma simulação de um quiz interativo. Ele foi desenvolvido com as tecnologias HTML, CSS e Javascript. Ele tem como objetivo de entreter o usuário e mostrar  seu aproveitamento!

## Tecnologias utilizadas
Essse projeto foi desenvolvido seguindo as seguintes tecnologias:

- HTML - Estrutura do layout da página (https://html.com/) 
- CSS - Estilização do layout (https://css.com/)
- JavaScript - Implementa a lógica de iniciar o quiz, mostrar cada pergunta e mostrar o resultado (https://www.javascript.com/)

## Funcionalidades
- Exibi o botão de iniciar o quiz
- Mostra cada pergunta com suas respectivas opções
- A cada pergunta exibida, pode-se selecionar uma alternativa e clicar em envia
- Depois de enviar a resposta da ultima pergunta, mostra o seu aproveitamento e o container dos resultados
  
## Como Usar
1. Abra o arquivo index.html em seu navegador da web.
2. Clique em iniciar Quiz
3. Selecione a melhor resposta para cada pergunta
4. Veja o seu aproveitamento no final
